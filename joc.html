Python 3.11

Python 3.11 just came out! the new version is around 25% faster on average during runtime. This is a huge improvement and will likely continue to grow in the near future.

Despite Python's new amazing runtimes, data scientists have run into a couple of problems with the new version. There aren't as many compatible binary packages as before, so certain files don't work as intended.

With these changes made to the new version of Python, programming has become more accessible towards children. They are able to toy with a little script on a laptop to give them a better understanding of computer science as a whole.

Even without the new Python 3.11 update, the job market flourishes with jobs available for python engineers. Wether it's data science, engineering, to AI development. Python is there.
